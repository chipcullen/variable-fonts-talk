Variable Fonts

What is a variable font?

“a single font file that behaves like multiple fonts”.
https://blog.typekit.com/2016/09/14/variable-fonts-a-new-kind-of-font-for-flexible-design/

What does this mean?

Multiple Weights…

…one file

It does mean variable.

Want a weight of 429?

How about 832?

209? Go for it.

we are no longer shackled to the conventional weights. we have much more subtle control.

How do you use a
Variable Font?

(On the web)


						@font-face {
							font-family: "Source Sans Pro";
							font-weight: 200 900;
							font-style: normal;
							src: url("../../SourceSansVariable-Roman.ttf.woff2") format("woff2"),
								url("../../SourceSansVariable-Roman.ttf.woff") format("woff"),
								url("../../SourceSansVariable-Roman.ttf") format("truetype");
						}
					

						@font-face {
							...
							font-weight: 200 900;
							...
						}
					

						h2 {
						  font-weight: 600;
						}
